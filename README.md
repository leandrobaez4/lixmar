# LIXMAR - Tienda de Electrónica en Línea

Implementación del diseño de Figma para LIXMAR, una tienda de electrónica moderna y responsiva.

## Estructura del Proyecto

```
lixmar/
├── index.html           # Página principal
├── styles/
│   └── main.css         # Estilos globales
├── src/
│   └── main.js          # JavaScript funcional
├── public/              # Carpeta para imágenes y assets
└── README.md            # Este archivo
```

## Características Implementadas

### Secciones del Sitio

1. **Header y Navegación**
   - Logo de LIXMAR
   - Menú de navegación principal
   - Iconos de búsqueda, cuenta y carrito
   - Banner promocional

2. **Sección Hero**
   - Bienvenida destacada
   - Imagen de producto destacado
   - Botón de llamada a acción

3. **Productos Destacados**
   - Grid responsivo de productos
   - Información de precio con descuentos
   - Sistema de calificaciones
   - Botón "Agregar al carrito"

4. **Categorías**
   - Grid de categorías de productos
   - Imágenes de categoría
   - Contador de productos

5. **Más Vendidos**
   - Carrusel horizontal de productos
   - Scroll personalizado

6. **Marcas**
   - Grid de logos de marcas asociadas

7. **Newsletter**
   - Suscripción a boletín informativo
   - Validación de email

8. **Footer**
   - Enlaces de información
   - Redes sociales
   - Métodos de pago
   - Copyright

### Funcionalidades JavaScript

- **Carrito de Compras**: Sistema de carrito con localStorage
- **Notificaciones**: Mensajes de éxito/error
- **Navegación Suave**: Desplazamiento suave entre secciones
- **Búsqueda**: Interfaz de búsqueda
- **Newsletter**: Suscripción con validación

## Estilos y Diseño

- **Colores principales**: 
  - Naranja (#ff6b35)
  - Morado gradiente (#667eea, #764ba2)

- **Tipografía**: Segoe UI y fuentes del sistema

- **Responsive Design**: 
  - Desktop: 1200px
  - Tablet: 768px
  - Mobile: 480px

## Instalación y Uso

1. Coloca los archivos en tu servidor web
2. Asegúrate de tener las imágenes en la carpeta `public/`
3. Abre `index.html` en tu navegador

## Imágenes Requeridas

Las siguientes imágenes deben colocarse en la carpeta `public/`:

- `hero-product.jpg`
- `product-1.jpg` a `product-4.jpg`
- `bestseller-1.jpg` a `bestseller-5.jpg`
- `cat-audio.jpg`, `cat-wearables.jpg`, `cat-photography.jpg`, `cat-mobile.jpg`
- `brand-1.png` a `brand-6.png`
- `payments.png`

## Navegadores Compatibles
